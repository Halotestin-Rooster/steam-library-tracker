About This App

This Game Library Tracker is built to let users authenticate with Google, verify email, and maintain a saved game profile across sessions.

Features

  • Gmail / Google sign-in and email/password signup with verification.
  • Automated Web Scraping & Data Mining using games API integrations.
  • Persistent account game progress saved in Firebase.
  • Game library with cover images, search, and played status.
  • Rating calculation based on selected played games.
  • Guest browsing mode with rating locked until login.

Publisher: AngryBird

How it works

When you sign in, the app stores your played game list in Firebase. You can mark games as played, search the library, and calculate a rating whenever you're logged in with a verified account.
